РЕЖИМ КЛИКЕР

На экране появляются 4 цветные цели, задача игрока - щелкать по ним мышью. Попадание дает +1 очко, а промах −1 очко.

РЕЖИМ ТРЕКИНГ

На экране появляется одна движущаяся цель, задача игрока удерживать курсор внутри цели. Пока курсор внутри – очки начисляются непрерывно в сумме 10 очков/сек. При выходе – рост счёта прекращается
